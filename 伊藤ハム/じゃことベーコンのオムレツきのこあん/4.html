じゃことベーコンのオムレツきのこあん

材料(4人分)

  • 朝のフレッシュ　ハーフベーコン ... 2パック
  • ちりめんじゃこ ... 15g
  • 冷凍ポテト（ナチュラルタイプ） ... 150g
  • 卵 ... 6個
  • だし汁(A) ... 大さじ3
  • マヨネーズ(A) ... 大さじ2
  • 塩・ｇしょう(A) ... 少々
  • 片栗粉(A) ... 小さじ1
  • えのき ... 1パック
  • にんじん ... 40g
  • 絹さや ... 12枚
  • 酒(B) ... 大さじ2
  • みりん(B) ... 大さじ1
  • 薄口醤油(B) ... 小さじ1/2
  • 塩(B) ... 大さじ1・1/2
  • 片栗粉(B) ... 大さじ1
  • サラダ油 ... 適量

作り方（調理時間:20分/目安）

  1. ベーコンは粗みじん切り、冷凍ポテトはざく切りにし、ちりめんじゃことともに溶き卵に加えＡを混ぜる。
  2. サラダ油を熱し、①を入れオムレツを4個作る。
  3. にんじんは紅葉型でぬき、残りは千切り、えのきは半分に切り、絹さやとともにＢでさっと煮、えのきあんを作る。
  4. ②に③をかける。

提供

伊藤ハム米久ホールディングス（株）
