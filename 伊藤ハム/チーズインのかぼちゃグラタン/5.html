チーズインのかぼちゃグラタン

材料（2人分）

  • チーズイン ... 1袋
  • かぼちゃ ... 1/8個
  • ブロッコリー ... 1/4個
  • 玉ねぎ ... 1/4個
  • にんじん ... 1/4個
  • 赤パプリカ ... 1/8個
  • マカロニ（ファルファーレ） ... 60g
  • 白ワイン ... 大さじ1/2
  • 牛乳 ... 50cc
  • バター ... 大さじ1
  • 塩コショウ ... 少々
  • 牛乳(A) ... 100cc
  • バター(A) ... 大さじ1/2
  • 小麦粉(A) ... 大さじ1/2
  • 塩コショウ(A) ... 少々

作り方（調理時間:35分/目安）

  1. かぼちゃは一口大に切り、レンジで4～5分加熱して潰す。玉ねぎは薄切り、ブロッコリーは小房に分ける。にんじんは薄切りにして塩ゆでし、型で抜く。赤パプリカはさいの目に切り塩ゆでしておく。
  2. Aを鍋に入れて弱火にかけ、ソースをつくる。とろみが出てきたらかぼちゃを加えてさらに混ぜる。
  3. 熱したフライパンにバターを入れて玉ねぎ・ブロッコリー・にんじんを炒め、白ワイン・塩コショウで調味する。
  4. ③に牛乳・ゆでたマカロニを入れてさっと煮る。②を半量混ぜる。
  5. 耐熱皿にバターをぬり④とかぼちゃ・赤パプリカ・斜め半分に切ったチーズインを盛り付け、オーブントースターで約10分焼き上げる。

提供
