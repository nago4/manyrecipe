野菜たっぷり和風煮込みハンバーグ

材料（1人分）

  • 旨包ボリュームリッチハンバーグ （和風タイプがお薦め） ... 2個
  • れんこん ... 4cm
  • 生しいたけ ... 3枚
  • にんじん（小さめ）... 1本
  • さつまいも（小さめ）... 1本
  • 三つ葉 ... 適量
  • 油 ... 大さじ1
  • だし汁 ... 1と1/2カップ
  • 砂糖 ... 大さじ1
  • 醤油、みりん ... 各大さじ2

作り方（調理時間:20分/目安）

  1. れんこん、にんじん、さつまいもは4つ割りにし、5㎜幅に切る。生しいたけは4つ切りにする。
  2. フライパンに油を熱し、①を炒める。
  3. Aを加え、れんこんが軟らかくなるまで煮、ハンバーグをソースごと加え2～3分煮る。
  4. 器にハンバーグをのせ、煮野菜を入れ、煮汁をかけ、三つ葉を散らす。

提供

伊藤ハム米久ホールディングス（株）
